# INICIAR EL REPO

Se hace una sola vez
```
git init
```

# AGREGAR UN ARCHIVO

Una sola vez por archivo
```
git add nombre_archivo
```

# GRABAR UN COMMIT
```
git commit -a -m comentario
```

# VER LISTA DE COMMITS
```
git log
```

# VER ULTIMO COMMIT
```
git show
```

# VER DIFERENCIAS CON ULTIMO COMMIT
```
git diff
```

# DESHACER UN COMMIT
```
git checkout nombre_archivo
```
